# Keybindings

This setup uses a keyboard-centric workflow, primarily relying on the `SUPER` key (Windows key/Command key).

## Core System

| Keybinding | Action |
| :--- | :--- |
| `SUPER + Q` | Close active window |
| `SUPER + ESCAPE` | Open Power Menu (wlogout) |
| `SUPER + RETURN` | Open Terminal |
| `SUPER + D` | Open App Launcher (Rofi/Fuzzel) |
| `SUPER + E` | Open File Manager |
| `SUPER + Z` | Open Web Browser |

## Window Management

| Keybinding | Action |
| :--- | :--- |
| `ALT + RETURN` | Toggle Fullscreen |
| `SUPER + SPACE` | Toggle Floating |
| `SUPER + P` | Toggle Pseudo Tiling |
| `SUPER + J` | Toggle Split |
| `SUPER + H/J/K/L` | Move focus (Left/Down/Up/Right) |
| `SUPER + SHIFT + H/J/K/L`| Move window (Left/Down/Up/Right) |

## Workspaces

| Keybinding | Action |
| :--- | :--- |
| `SUPER + [1-9]` | Switch to workspace [1-9] |
| `SUPER + SHIFT + [1-9]` | Move active window to workspace [1-9] |
| `SUPER + Scroll` | Scroll through workspaces |

## Media & Audio

| Keybinding | Action |
| :--- | :--- |
| `XF86AudioRaiseVolume` | Volume Up |
| `XF86AudioLowerVolume` | Volume Down |
| `XF86AudioMute` | Mute/Unmute Audio |
| `XF86AudioPlay` | Play/Pause Media |

## Notifications & Utilities

| Keybinding | Action |
| :--- | :--- |
| `SUPER + N` | Toggle Notification Center |
| `SUPER + SHIFT + N`| Close current notification |
| `SUPER + SHIFT + F`| Open File Search |
| `SUPER + W` | Change Wallpaper |

> Note: You can view all active keybindings directly in the UI by pressing `SUPER + SHIFT + /` or viewing the `configs/hypr/configs/keybinds.lua` file.
